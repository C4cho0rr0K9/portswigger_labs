## 🚀 Repository Objective
The goal of this project is to build a comprehensive knowledge base of web vulnerabilities. Each writeup includes:
* **Vulnerability Analysis:** Understanding the "why" and "how".
* **Methodology:** Step-by-step reproduction using Burp Suite.
* **Exploitation:** Final payloads and logic bypasses.
* **Remediation:** How to patch the flaw.
